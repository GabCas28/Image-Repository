# Initial classes

The initial classes are the following:

* [Picture](../src/Picture.ts) (Main entity)
  * Source: String
  * Author: User
  * Title: String
  * Description: String
  * Comments : Comment List
  * Ratings : Rating List
  
* [Rating](../src/Rating.ts)
  * Author: User
  * Value: Integer
  
* [Review](../src/Review.ts)
  * Author: User
  * Title: String
  * Body: String
  * Votes: Integer

* [User](../src/User.ts)
  * Name: String
  * Email: String

* [Controller](../src/Controller.ts)
  * ListPictures: Picture List
